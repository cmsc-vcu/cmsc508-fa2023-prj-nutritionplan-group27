CMSC 508 Final Project
Nutrition Database

This website is the simple frontend for our semester long database project. We chose to make a nutrition and recipe database because it aligned with all of our team members' interests.

Currently our database holds over 15,000 recipes with each of the recipes linked to the mealplans that fit it best using a junction table with over 33,000 entries.

Our website allows its users to add a goal that fits them best, join a meal plan that fits them best and search the wide variety of recipes that are in that meal plan.
